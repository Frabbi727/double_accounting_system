## ⚠️ তিনটা জরুরি কথা

**১. কোড চালিয়ে পরীক্ষা করা হয়নি।**
সব PHP ফাইলের syntax পরীক্ষা করা হয়েছে (সবগুলো পাস), কিন্তু যে পরিবেশে লেখা হয়েছে সেখানে Laravel/Composer ছিল না। তাই প্রথমবার `php artisan test` চালালে ছোটখাটো সমস্যা আসতে পারে (namespace path, টেস্টে User factory ইত্যাদি)। এগুলো সহজেই ঠিক হবে।

**২. কাট-অফ তারিখ আগে সেট করুন।**
`.env` ফাইলে `SHOP_CUTOFF_DATE` — কোনো ডেটা ঢোকানোর **আগে**। প্রতিটা ওপেনিং এন্ট্রি ওই তারিখে বসবে। পরে বদলাতে গেলে সব ওপেনিং রিভার্স করতে হবে।

**৩. ক্রয় মডিউল আগে, বিক্রয় পরে।**
স্টক না থাকলে বিক্রয় টেস্ট করা যায় না। ক্রয় দিয়ে স্টক ঢোকানো শিখলে বিক্রয় বানানো সহজ হবে।

## দ্রুত শুরু

```bash
composer create-project laravel/laravel shop-accounts
cd shop-accounts
composer require laravel/breeze --dev
php artisan breeze:install blade
composer require spatie/laravel-permission barryvdh/laravel-dompdf

# 03-code/ ফোল্ডারের app/ config/ database/ tests/ docs/ কপি করুন

echo "SHOP_CUTOFF_DATE=2026-07-31" >> .env   # ★ ডেটা ঢোকানোর আগে সেট করুন

php artisan migrate
php artisan db:seed --class=ChartOfAccountsSeeder
php artisan test --filter=OpeningBalanceTest
```

শেষ কমান্ডটা পাস করলে বুঝবেন ভিত্তি ঠিকমতো দাঁড়িয়েছে।
